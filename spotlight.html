Welcome to the Spotlight!

Got an awesome side project that you've been working on?
Looks like it's time to show it off.

Here you can submit your program which demonstrates your knowledge of programming in any area of choice.

  • These programs will generally be side projects and fulfill some purpose - entertainment, convenience, or simply to show off skills.
  • The programs will be primarily scored based on their creativity and good code design and implementation.
  • Spotlight submissions are evaluated every two months for a winner.
The rules are simple:
  • Code must be submitted for the current Spotlight (see Submission tab).
  • Code must be accessible on some online repository (such as GitHub).
  • Project should generally not be a school assignment.
  • Can feature any coding language with almost any software for your project (Java, Angular, Visual Studio, etc).
That’s it. The prize will be a $25 Amazon gift card; but more importantly, you will be added into the Hall of Fame as a Spotlight winner for peers and companies alike to see. And remember: even if you don’t win, a good program will always make it for honorable mention. So make sure to get your submission in! A new form for submission is in the works right now.
